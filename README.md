#### Time complexity
Involves the measure of time used to run an algorithm as the input size increases

#### Constant Time - O(1)
Has the same running time regardless of the size of the input

For Example:

   func checkFirst(names:[String]){
    if let first = names.first {
        print(first)
    } else {
        print("no names")
    }
}
